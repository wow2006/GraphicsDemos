# QuadTree
is a tree data structure in which each internal node has exactly four children.

## Types
- region quadtree.
- point quadtree.
- point-region (PR) quadtree. [Implementation](pointRegionQuadtree.cpp)
- edge quadtree.
- polygonal-map (PM) quadtree.
- compressed quadtree.
